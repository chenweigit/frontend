# nodejs 链接mysql
--------------------------

## nodejs 核心模块没有mysql模块，需要第三方模块。
   1. 为node.js安装mysql模块使用node.js中的npm管理器。
   2. cmd，进入你的项目的路径。
   3. 在命令行输入 npm install mysql 之后npm会自动为你下载Mysql模块默认放在你的项目路径下。下载完成后会生成node_modules的目录。里面是什么暂时可以不用去管理。
